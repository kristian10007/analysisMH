# Preparation
- Copy the Exel-files to the according folder.
  - "Doktorarbeit_Hilke_8.3.xslx" to folder "Hilke3"
  - "Tabelle_mit_Patientennummer_K.xslx" to folder "Michelle3"

- Install docker from https://www.docker.com

# Running the analysis
- Open a terminal
- Navigate in the terminal to the folder containing this file.
- Execute one of this commands:
  docker-compose up michelle
  docker-compose up hilke
